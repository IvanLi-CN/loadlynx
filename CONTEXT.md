## Glossary

### 总览

总览是 Web Console 的一级导航入口，面向多设备观察与切换。它复用根路径 `/` 与 `/devices`，展示所有已知设备的核心状态、连接方式，以及进入当前设备 `仪表盘` 或 `系统` 的动作。

### 仪表盘

仪表盘是当前设备的主工作面，对应“信息展示 + 控制”的同屏界面。它继续复用 `/$deviceId/cc` 深链，保留仪器风格的左监右控布局，并吸收完整的 USB-PD 查询、校验与 Apply 流程。

### 系统

系统是当前设备的管理域一级导航，承载 `设置 / 校准 / 状态 / 固件 / 关于` 二级页面。它描述的是当前设备的配置、维护、诊断与身份信息，而不是多设备列表。

### 设备抽屉

设备抽屉是桌面端当前设备切换入口的右侧 sheet。它只负责展示设备列表、连接方式与切换动作，不承担一级导航。

### 连接方式

连接方式是设备卡片和设备切换入口中展示的 transport 摘要。它由前端本地已知事实推导，包括 `connectionMarks`、`devd`、`webSerial` 与 `baseUrl`，用于帮助用户判断当前设备通过 LAN、USB、Web Serial 或 devd 路径接入。
